# Charts Implementation

## Overview
This implementation includes interactive charts using **Highcharts** to display stock performance data on the Dashboard page.

## Features

### 1. Stock Price Trend Chart (Line Chart)
- Displays stock price movements over the last 30 days
- Interactive tooltips showing date and price
- Smooth animations
- Responsive design

### 2. Volume Analysis Chart (Column Chart)
- Shows trading volume with column bars
- Overlays gain/loss percentage as a line chart
- Dual Y-axis for volume and percentage
- Color-coded for better visualization

## Technologies Used

- **Highcharts** (v12.4.0) - Professional charting library
- **highcharts-react-official** (v3.2.3) - Official React wrapper
- **TanStack Query** (v5.90.10) - Data fetching and caching
- **Material-UI** (v7.3.5) - UI components

## Data Source

Currently using **mock data** generated by `src/services/mockStockData.ts`:
- `fetchStockPriceData()` - Returns 30 days of price data
- `fetchVolumeData()` - Returns 30 days of volume and gain/loss data

The data is fetched using TanStack Query for:
- Automatic caching
- Background refetching
- Loading states
- Error handling

## How to Use

The charts are automatically loaded on the Dashboard page. They use TanStack Query to fetch and cache data, with automatic loading indicators while data is being fetched.

To modify the data source:
1. Update `src/services/mockStockData.ts` to use a real API
2. The charts will automatically use the new data structure
